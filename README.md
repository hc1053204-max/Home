# 組別: 23
## 遊戲規則
關卡數量： 共 3 個關卡。
通關條件： 玩家必須在每一關中擊敗所有小怪，並成功擊敗該關卡的最終 Boss，才能進入下一關。
最終勝利條件： 完成第 3 關並擊敗最終 Boss，即可獲得遊戲勝利。
失敗條件： 當主角的 HP（生命值）降至 0 時，角色死亡，遊戲立即結束。
關卡進行方式： 關卡必須依序挑戰，無法跳過未完成的關卡。
敵人設定： 每個關卡皆配置不同數量與能力的小怪，以及一名擁有較強戰鬥能力的 Boss。
遊戲流程
進入第 1 關並消滅所有小怪。
挑戰並擊敗第 1 關 Boss。
依序完成第 2、3 關的挑戰。
擊敗第 3 關最終 Boss 後，遊戲通關。
若主角 HP 歸零，則立即判定遊戲失敗。
